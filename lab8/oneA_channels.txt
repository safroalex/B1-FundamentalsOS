    Создание канала: pipe(filedes) создает канал с двумя концами — для чтения (filedes[0]) и для записи (filedes[1]).

    Создание дочернего процесса: fork() порождает новый процесс, который является копией текущего. В дочернем процессе fork() возвращает 0, в родительском — ID дочернего процесса.

    Дочерний процесс:
        Закрывает конец канала для чтения (filedes[0]).
        Открывает файл input.txt для чтения.
        Читает символы из файла и записывает их в канал.
        Закрывает файл и конец канала для записи.

    Родительский процесс:
        Закрывает конец канала для записи (filedes[1]).
        Открывает файл output.txt для записи.
        Читает символы из канала и выводит их на экран и в файл output.txt.
        Закрывает файл.

    Завершение: Оба процесса завершают выполнение.

Этот код осуществляет простую задачу перенаправления данных из одного файла в другой через механизм каналов в модели родитель-ребёнок.
